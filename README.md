Este proyecto es mi portafolio web personal, diseñado para mostrar mis habilidades, tecnologías y proyectos desarrollados en el área de software.

La página está construida con un enfoque moderno y minimalista, destacando tanto el diseño visual como la estructura lógica del contenido. 
Incluye secciones como presentación personal, aprendizaje continuo y proyectos relevantes, además de opciones de contacto.
Portafolio web estático desarrollado con HTML, CSS y JavaScript.
